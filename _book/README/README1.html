学分说明

学分说明

2019年7月之前，人文项目作为“知”，与成长中心负责的社会实践“行”，记为校本必修。每项目2学分，2021届所有学生在高一高二两年需获取4学分； 因2019年9月之后研学转接到人文项目内，本届研学学分不足的同学，需申报人文项目或依照《北大附中项目式学习/研究性学习评价标准》自主申报，以满足研学6学分要求。

2019年9月之后入学新生，人文项目与研究性学习合并，记为国家必修。各项目调整学时后设置3学分，所有学生在高一高二两年需修习6学分。同时，公共说理作为全校学生的国家必修。 人文项目/研究性学习，记学分，不计积点。

为什么

降低高于国家的学分要求，是其一；学理上，研学即项目，是其二。
