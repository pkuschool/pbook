# 人文项目

为落实附中培养目标中公民教育，落实项目式学习研发，2018年，北大附中启动了人文项目。

迄今日，人文项目包含公共说理、模拟法庭、书院视点、银杏讲坛、社会创新，北大附中口述史六个子项目。

---

公民教育的落实，除各个项目选题展开围绕公民事务之外，学院会发布北大附中年度公民教育报告。

项目式学习研发，呈现于本网站项目运营资料和本网站各项目月度总结。
